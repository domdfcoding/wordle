****************
wordle
****************

.. start short_desc

**Create a wordcloud for a Git repository.**

.. end short_desc

Can also create wordclouds from directories of source files or a single source file.

``wordle`` can be installed from PyPI.

To install with ``pip``:

.. code-block:: bash

	$ python -m pip install wordle

.. end installation
